{% extends "_page.html" %} {% block intro %} Error handling in the three languages. {% endblock %} {% block description %} C++ and Java both have exception throwing and handling capabilities, Rust does not. Instead Rust handles errors through its return type, commonly using Result<T, E>. The try-catch is replaced with a match on Ok and Err.

One should note that in Rust the f64 primitive type was used everywhere; whereas in C++ and Java the respective compilers did not complain about the conversion from int to double. The Rust compiler would not allow similar code to compile. {% endblock %}
